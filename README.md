# Photo_take
零件連續拍攝與自動儲存系統 (Photo Capture System)

本專案透過 Google Colab 結合網頁前端 JavaScript 技術，實現利用電腦視訊鏡頭（或搭配外接虛擬相機 App）進行定時、定量的連續零件照片拍攝，並將照片自動同步儲存至使用者的 Google 雲端硬碟 (Google Drive) 中，適合用於工業零件數據集收集與品質檢測影像存檔。
🛠 智慧型手機鏡頭連接指南 (使用 iVCam)

為了獲得更高畫質的零件照片，建議使用智慧型手機的後置鏡頭代替筆電內建鏡頭。本專案推薦並支援使用 iVCam 進行連線。
步驟 1：下載與安裝

    電腦端 (Windows)： 請至 e2eSoft iVCam 官方網站 下載並安裝 Windows 用戶端軟體。

    手機端 (Android / iOS)： 請在 App Store 或 Google Play 商店搜尋並下載 iVCam App。

步驟 2：裝置連線

    將你的電腦與手機連線至同一個 Wi-Fi 網域（或者使用 USB 傳輸線連接電腦與手機，傳輸更穩定）。

    先開啟電腦端的 iVCam 軟體。

    開啟手機端的 iVCam App，軟體會自動搜尋並完成連線。此時，你可以在電腦畫面上看到手機相機的即時預覽畫面。

    (建議) 在手機端設定中，將拍攝焦距固定、並調整至最高的解析度 (如 1080p 或 4K)，以利零件細節留存。

    檢查 iVCam 電腦端軟體畫面（最直覺）

    未連線成功： 開啟電腦上的 iVCam 軟體，如果畫面顯示「Waiting for iVCam connection...」（等待 iVCam 連線），代表尚未連上手機。

    連線成功： 如果連線成功，電腦上的 iVCam 視窗會立刻直接顯示手機相機拍到的即時畫面。此時手機畫面的右下角或上方也會顯示連線圖示。

2. 檢查 Windows「裝置管理員」（確認系統有抓到驅動）

有時候軟體卡住，你可以確認 Windows 系統底層有沒有把手機識別為視訊鏡頭：

    在 Windows 開始選單（或工作列放大鏡）搜尋 「裝置管理員」 並開啟。

    尋找 「影像裝置」（Cameras 或 Imaging devices）這一欄並展開。

    如果有看到 e2eSoft iVCam，代表 Windows 系統已經成功將它識別為一個可用的虛擬視訊鏡頭了！

3. 使用 Windows 內建的「相機」App 測試

這是最保險的驗證方式，能確保其他軟體（如瀏覽器、Colab）也能讀到它：

    在 Windows 搜尋列輸入 「相機」，打開 Windows 內建的相機應用程式。

    如果你的電腦有內建鏡頭，它可能會先顯示內建畫面。請點擊相機 App 右上角或側邊的 「切換相機」按鈕（通常是一個相機帶有旋轉箭頭的圖示）。

    切換後，如果畫面成功轉成手機鏡頭拍到的畫面，那就百分之百確定連線成功且可以正常使用了！

🚀 使用步驟與操作說明

請在 Google Colab 環境中依序執行以下儲存格：
1. 基礎初始化與路徑建立

執行第一個儲存格以在 Google Drive 中建立預設的相片儲存資料夾。
2. 雲端硬碟掛載與環境設定

執行主程式時，Colab 會跳出權限請求視窗：

    請允許 掛載 Google Drive 權限。

    瀏覽器會跳出要求攝影機使用權限，請務必點選 「允許」。

3. 模式選擇與拍攝

本程式提供兩種連續拍攝模式，你可以根據需求選擇執行：
🔹 模式 A：定時定量拍攝 (指定總時間)

    程式位置： 第二個程式儲存格。

    自訂參數： * DURATION: 總拍攝秒數 (預設 23 秒)。

        INTERVAL: 每次拍照間隔秒數 (預設 2.0 秒/張)。

    操作： 點擊執行後，畫面上方會出現即時預覽視窗，並倒數計時自動拍照。亦可隨時點擊 🛑 停止拍攝 提前結束。

🔹 模式 B：特定鏡頭鎖定拍攝 (推薦 iVCam 使用)

    程式位置： 第四個程式儲存格 (核心功能)。

    特點： 此模式會自動搜尋名為 iVCam 的外接相機裝置，避免 Colab 誤用筆電內建的前置鏡頭。

    自訂參數：

        TARGET_CAMERA_NAME: 設為 "iVCam"。

        MAX_IMAGES: 預期拍攝的總張數 (預設 5 張)。

        INTERVAL: 每次拍照間隔秒數 (預設 2.0 秒)。

    ⚠️ 第一次執行注意： 模式 B 在啟動時會先短暫喚醒預設鏡頭以向瀏覽器取得完整的鏡頭清單（解決裝置名稱為空白的問題），接著會自動切換至 iVCam 鏡頭。請在畫面上看到手機畫面對準零件後再開始移動或更換零件。
